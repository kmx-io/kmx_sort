# kmx.io/kmx_sort v0.2

Portable sort order using memcmp and strlen.

This is a stable release, for a development branch see
[kmx.io/kmx_sort](https://git.kmx.io/kmx.io/kmx_sort/)

## Usage

```sh
$ sort [-u]
```

## Installation

```sh
./configure && make
```

As root :
```sh
make install
```
